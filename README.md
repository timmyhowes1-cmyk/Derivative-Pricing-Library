This is a derivative pricing library I have been working on that I started from scratch. It is designed so that new models, instruments and numerical schemes can be added at ease. Analytical solutions for prices/greeks will be implemented where possible, otherwise Monte Carlo simulation can be used.

Current models implemented: BSM, Heston, CIR

Current instruments implemented: European, Asian, Lookback, Digital, Barrier

Current pricing/greeks engines implemented: BSM Analytical, Heston Analytical, Monte Carlo

More to be included over time...
